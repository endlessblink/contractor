# Purpose

Create professional Israeli-market CVs in Hebrew, with correct RTL structure, concise positioning, and readable mixed Hebrew/English technical content.

# CV Principles

- Keep the CV direct, credible, and achievement-oriented.
- Prefer concrete experience, projects, tools, clients, teaching, publications, and community activity over generic self-description.
- Do not use inflated corporate language.
- Do not write in first person.
- Avoid long paragraphs except for a short profile summary.
- Preserve English names of tools, products, models, repositories, and links.

# Recommended Section Order

1. Header: full name, role/headline, location.
2. Contact: phone, email, LinkedIn, portfolio, GitHub.
3. Profile: short paragraph, 3-5 lines.
4. Teaching and training, if relevant.
5. Professional experience.
6. Open-source tools and products.
7. Selected projects and exhibitions.
8. Community activity and publications.
9. Skills and tools.
10. Languages.

# Header Rules

- Name should be the strongest visual element.
- Headline should describe the candidate clearly in one line.
- Contact links should be compact and readable.
- Do not bury portfolio or GitHub links if they are important to the role.

# Experience Rules

- Each role should include role title, organization/client/context, and date range.
- Bullets should describe output, scope, tools, clients, or measurable value.
- Keep bullets short enough to scan quickly.
- For freelance work, group related work under a clear business or independent heading.

# Skills Rules

- Group skills by domain, not as one long unstructured list.
- Keep English tool names in English.
- Prioritize skills that support the target role.
- Remove weak or unrelated tools if they dilute the profile.

# Israeli CV Style

- Hebrew CVs can be direct and practical; avoid cover-letter language.
- A one-to-two page CV is preferable unless the experience genuinely requires more.
- Include language proficiency near the end.
- Include community/public work when it strengthens professional credibility.
